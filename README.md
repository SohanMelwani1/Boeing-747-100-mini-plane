# Boeing-747-100-mini-plane
A mini airplane of Boeing 747-100 (I tried adding as much details as possible, however there were some small changes)

instructions:
Print the body and engine using PLA, But use PETG for the wheels for easier movement of the plane

Glue The middle, top and bottom of the plane together (make sure that it is alligned before gluing it together and use plastic glue).

Once glued, Take the engines and place them in thier slots and cover them with the clip to stop them from moving (you can glue the clips together however it is optional)

Attach each wheel to the plane (all wheels have the same size so it's easy to replace) However before attaching you can sand the wheels for a smoother flight of the ground. 


Print settings:

10-15% infill 

Add support for over 45 degrees

use .2mm for initial height thickness and .12 for layer height after initial layer height for a better finish.

(I am printing it with Bambu labs A1 mini with .4mm nozel)
